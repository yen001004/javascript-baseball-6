## 📌 구현할 기능 목록

### 🖥️ 입출력 구현

- [x] 프롬프트 띄우기
- [x] 숫자 입력받기
- [x] 숫자 판별 후 스코어 출력하기
- [x] 3개를 모두 맞출 경우 종료 프롬프트와 함께 1 또는 2 입력받기

### ⚾️ 기능 구현

- [x] 하나의 랜덤값 구하기
- [x] 볼, 스트라이크 판별하기
  - [x] 다른 자리에 같은 수가 있다면 볼
  - [x] 같은 자리 같은 수가 있다면 스트라이크

### 🤔 예외 처리

- [x] 1-9까지의 세 자리 숫자가 아닌 값 입력
- [x] 중복되는 숫자를 포함한 입력
- [x] 종료 시 1 또는 2가 아닌 값 입력
